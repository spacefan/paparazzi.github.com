12 Paparazzi is an attempt to develop a free software Unmanned (Air) Vehicle System.
13  As of today the system is being used successfuly by a number of hobbyists, universities and companies all over the world, on vehicle of various size ( 100g to 25Kg ) and of various nature ( fixed wing, rotorcrafts, boats and surface vehicles).
14 
15 Up to date information is available in the wiki http://wiki.paparazziuav.org
16 
17 and from the mailing list [paparazzi-devel@nongnu.org] (http://savannah.nongnu.org/mail/?group=paparazzi)
18 and the IRC channel (freenode, #paparazzi).

21 Required Software
22 -----------------
23 
24 Installation is described in the wiki (http://wiki.paparazziuav.org/wiki/Installation).
25 
26 For Ubuntu users, required packages are available in the [paparazzi-uav PPA] (https://launchpad.net/~paparazzi-uav/+archive/ppa),
27 Debian users can use the [OpenSUSE Build Service repository] (http://download.opensuse.org/repositories/home:/flixr:/paparazzi-uav/Debian_7.0/)
28 
29 Debian/Ubuntu packages:
30 - **paparazzi-dev** is the meta-package that depends on everything needed to compile and run the ground segment and the simulator.
31 - **paparazzi-jsbsim** is needed for using JSBSim as flight dynamic model for the simulator.
32 
33 Recommended cross compiling toolchain: https://launchpad.net/gcc-arm-embedded

50 Compilation and demo simulation
51 -------------------------------
52 
53 1. type "make" in the top directory to compile all the libraries and tools.
54 
55 2. "./paparazzi" to run the Paparazzi Center
56 
57 3. Select the "Microjet" aircraft in the upper-left A/C combo box.
58  Select "sim" from upper-middle "target" combo box. Click "Build".
59  When the compilation is finished, select "Simulation" from
60  the upper-right session combo box and click "Execute".
61 
62 4. In the GCS, wait about 10s for the aircraft to be in the "Holding point" navigation block.
63  Switch to the "Takeoff" block (lower-left blue airway button in the strip).
64  Takeoff with the green launch button.
65 
66 Uploading of the embedded software
67 ----------------------------------
68 
69 1. Power the flight controller board while it is connected to the PC with the USB cable.
70 
71 2. From the Paparazzi center, select the "ap" target, and click "Upload".
