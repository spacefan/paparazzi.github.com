3506 ## SHAPE (ID 38) ## {#SHAPE}
3507 
3508 The SHAPE message used to draw shapes onto the Paparazzi GCS.
3509  Field name shape is used to define the type of shape i.e. Circle, Polygon, Line, or Text.
3510  This is indexed from 0-3 respectively.
3511 
3512  Each shape drawn must have an id number associated with it.
3513  This id number in conjuction with the shapetype will be needed to update or delete the shape.
3514  A circle can be defined with the same id as a polygon but since they have different shape types they are considered unique.
3515 
3516  linecolor and fillcolor take in a color string ie: "red", "blue"
3517 
3518  opacity will change the level of transparency of the fill.
3519  0 - Transparent
3520  1 - Light Fill
3521  2 - Medium Fill
3522  3 - Opaque
3523 
3524  Passing a status of 0 will create or update the shape specified by id and type.
3525  Passing a status of 1 will delete the shape specified by id and type.
3526 
3527  latarr is an array of coordinates that contain the latitude coordinate for each point in the shape.
3528  The array is comma separated.
3529  lonarr is similar to latarr but contain the longitude coordinate for each point in the shape.
3530 
3531  Circle and Text type will take the first coordinates given to place the shape.
3532  Polygon will take all the coordinates given.
3533  Line will take the first two coordinates given.
3534 
3535  Radius is only used for the circle.
3536 
3537  Text will always be populated with each message using the first set of coordinates.
3538  The text field can not be blank or have spaces.
3539  If text is not desired for a shape then pass "NULL" into the text field.
