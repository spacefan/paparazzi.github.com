Module configuration options

Configure Options

  • name: ADC_IR1 value: ADC_1
    Description: ADC for IR horizontal channel 1
  • name: ADC_IR2 value: ADC_2
    Description: ADC for IR horizontal channel 2
  • name: ADC_IR_TOP value: ADC_0
    Description: ADC for IR vertical channel
  • name: ADC_IR_NB_SAMPLES value: 16
    Description: number of samples

Module functions

Init Functions

These initialization functions are called once on startup.

  • infrared_adc_init()

Periodic Functions

These functions are called periodically at the specified frequency from the module periodic loop.

  • infrared_adc_update()
    • Frequency in Hz: 60.
    • Autorun: LOCK
      Periodic function automatically starts after init and can't be stopped.
