Detailed Description

|STX|length|...

payload=(length-4) bytes ...|Checksum A|Checksum B|

where checksum is computed over length and payload:

mora_ck_a = mora_ck_b = length
for each byte b in payload
mora_ck_a += b;
mora_ck_b += mora_ck_a;
